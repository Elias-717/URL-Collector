URL Collector (Chrome Extension)

Save URLs from the current tab or manual input. Stored links persist in local storage and open in new tabs.

Features
- Save current tab URL
- Save a typed URL
- Delete all saved URLs (double-click)
- Local storage persistence

Install (Developer Mode)
1. Open Chrome and go to chrome://extensions.
2. Enable Developer mode (top right).
3. Click "Load unpacked" and select this folder.

Mobile Install Note
This version can be installed from the Chrome extension download on your phone, and it is more flexible for the mobile viewport.

Usage
- Click the extension icon.
- Use "SAVE TAB" or "SAVE INPUT" to add links.
- Double-click "DELETE ALL" to clear.

Project Structure
- index.html: popup UI
- index.css: styles
- index.js: logic
- manifest.json: extension manifest
